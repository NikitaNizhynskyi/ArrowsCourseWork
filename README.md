# ArrowsCourseWork

2D puzzle game created in Unity.

## How to play

Download the latest Windows build from the Releases section.

1. Open Releases.
2. Download `Arrows_Final.zip`.
3. Extract the archive.
4. Run `ArrowsCourseWork.exe`.

## How to open the project

1. Download or clone the repository.
2. Open the project in Unity Hub.
3. Use the Unity version specified in `ProjectSettings/ProjectVersion.txt`.
